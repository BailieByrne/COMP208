Requirements:
- Have Python 3.12 installed
- In terminal type "pip install numpy"
- The deps folder sotres dependencies


To compile:

1. mkdir build
2. cd build
3. cmake ..
4. cmake --build . --config Release 

This will create an exe or .out dependant on your OS in the build\Release\stock_sim.(exe/out)
Run this through the terminal using .\Release\stock_sim.(exe/out)
