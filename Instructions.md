# Compilation Guide

## C++

### Windows
```bash
cmake --build build --config Release
./build/bin/Release/stock_sim.exe
```

### macOS
```bash
cmake --build build --config Release
./build/bin/Release/stock_sim
```

### THEN FOR THE JAVA BUILDING:

### Windows
```bash
./compile.bat
```

### macOS
```bash
bash compile.sh
```
